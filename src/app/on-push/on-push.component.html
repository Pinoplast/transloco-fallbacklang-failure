Structural Directive

  • Regular: {{ t('home') }}
  • With params: {{ t('alert', { value: dynamic }) }}
  • With translation reuse: {{ t('a.b.c') }}
  • Message format no res: {{ t('myPluralRule', { myCount: 0}) }}
  • Message format plural: {{ t('myPluralRule', { myCount: 5}) }}

Fallback with parameters

  • Fallback: {{ 'empty' | transloco: { value: param } }}
